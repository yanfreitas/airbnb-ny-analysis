# airbnb-ny-analysis
Ánalise exploratória de dados baseado em um dataset sobre o airbnb em Nova York no ano de 2019.
Foram utilizados no projeto:
Linguagem de programação - python
Libs - matplotlib, seaborn, pandas
Ambiente - Jupyter notebook

dataset link: https://www.kaggle.com/dgomonov/new-york-city-airbnb-open-data/kernels
